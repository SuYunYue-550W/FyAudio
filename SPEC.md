# FyAudio 产品规格书 v1.0

## 1. 产品概述

### 1.1 产品名称与定位

**FyAudio** - 多终端WiFi蓝牙同步音频播放系统

定位：面向家庭和小型办公场景的多房间音频同步解决方案。无需专业设备，利用现有WiFi网络和普通智能设备即可实现多房间音乐同步播放。

### 1.2 核心价值主张

1. **零门槛多房间音乐**: 无需专业功放和线缆，任意设备秒变同步音响
2. **全平台覆盖**: Windows/Android/macOS/树莓派统一体验
3. **亚秒级同步**: 多设备播放时间差控制在50ms以内
4. **灵活扩展**: 从2台到20+设备，按需扩展

## 2. 功能规格

### 2.1 核心功能

| 功能 | 优先级 | 描述 |
|------|--------|------|
| WiFi音频广播 | P0 | 将音源设备系统音频广播到局域网 |
| WiFi音频接收 | P0 | 接收并播放来自音源的音频流 |
| 设备发现 | P0 | 自动发现局域网内所有FyAudio设备 |
| 实时同步 | P0 | 基于时钟同步的多设备时序对齐 |
| 蓝牙输出 | P1 | 通过蓝牙A2DP连接无线音响 |
| 音量控制 | P1 | 独立控制各设备音量 |
| 手动延迟补偿 | P2 | 手动调整±200ms补偿硬件延迟 |

### 2.2 用户角色

| 角色 | 设备类型 | 功能范围 |
|------|----------|----------|
| **Source (音源)** | Windows/macOS/Android | 采集系统音频，广播到网络 |
| **Receiver (接收)** | 所有平台 | 接收音频流并播放 |
| **Gateway (网关)** | 树莓派 | 接收+蓝牙输出+WiFi中继 |

### 2.3 平台支持

| 平台 | 角色 | 状态 | 备注 |
|------|------|------|------|
| Windows 10/11 | Source/Receiver | 开发中 | 需VB-Audio Virtual Cable |
| Android 10+ | Source/Receiver | 开发中 | 需MediaProjection权限 |
| macOS 12+ | Source/Receiver | 规划中 | 需BlackHole虚拟声卡 |
| 树莓派 (Raspbian) | Gateway | 开发中 | ALSA + BlueALSA |

## 5. 交互规格

### 5.1 用户操作流程

**首次使用**:
1. 下载并安装对应平台客户端
2. 启动应用 → 自动搜索局域网设备
3. 选择一台设备设为音源
4. 其他设备自动接收播放

**音源切换**:
1. 在任意设备点击"设为音源"
2. 当前音源自动降级为接收
3. 切换无断连，延迟<1秒

**蓝牙连接**:
1. 切换到"蓝牙"Tab
2. 点击"扫描"
3. 选择配对设备并连接
4. 音频自动路由到蓝牙设备

### 5.2 音量控制

- **本地音量**: 仅影响本机播放音量
- **全局音量同步**: 音源端音量变化时，可选同步到所有接收端
- **蓝牙音量**: 跟随系统蓝牙A2DP音量

### 5.3 延迟补偿

- **场景**: 不同蓝牙音响有不同的硬件延迟
- **操作**: 在"同步"Tab拖动滑块调整
- **范围**: ±200ms，步进1ms
- **保存**: 自动保存到本地配置

## 6. 性能目标

| 指标 | 目标 | 备注 |
|------|------|------|
| 同步精度 | <50ms | 同一帧在各设备播放时间差 |
| 端到端延迟 | 30-100ms | 音源→接收的延迟 |
| 启动时间 | <3秒 | 从点击到播放 |
| 设备发现 | <5秒 | 局域网扫描时间 |
| 切换延迟 | <1秒 | 音源切换时间 |
| CPU占用(Source) | <15% | 单核占用 |
| 内存占用 | <100MB | RSS |
| 功耗影响 | <5% | 电池设备 |

## 7. 兼容性要求

### 7.1 Windows
- Windows 10 (1903+) / Windows 11
- VB-Audio Virtual Cable
- 建议 4GB RAM / 双核CPU

### 7.2 Android
- Android 10 (API 29) 及以上
- 蓝牙 A2DP 支持
- WiFi 802.11g+ 网络

### 7.3 树莓派
- 树莓派 3B+ / 4B
- Raspbian Bullseye (11)
- BlueALSA + ALSA

### 7.4 网络
- IPv4 局域网
- UDP多播/广播支持
- 建议千兆路由器

---

## 8. 已知限制

1. **iOS端**: 仅支持接收端，暂无系统音频采集方案
2. **视频同步**: 音频同步，视频唇音同步暂不支持
3. **离线使用**: 必须同一局域网，不支持公网
4. **加密**: 暂不支持加密传输（同一局域网可信）
5. **设备上限**: 建议不超过20台同局域网设备
