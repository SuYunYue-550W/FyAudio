- [x] `lib/services/bluetooth_service.dart` - Windows蓝牙A2DP
- [x] `lib/services/sync_service.dart` - Dart端同步引擎
- [x] `lib/screens/home_screen.dart` - 主界面（4 Tab）

- [x] `lib/widgets/sync_indicator.dart` - 同步状态指示器
- [x] `lib/widgets/role_switcher.dart` - 角色切换器
